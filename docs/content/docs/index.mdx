Lazyskills is a terminal UI for browsing and managing reusable AI coding skills.

## Quick Start

1. Install dependencies for this repository.
2. Run `just docs-sync` to generate fresh reference pages from Rust source comments.
3. Run `just docs` and open `/docs`.

## Core Sections

- `getting-started`: first-run setup and local development.
- `core`: architecture and workflow design details.
- `reference`: generated API and module docs.
