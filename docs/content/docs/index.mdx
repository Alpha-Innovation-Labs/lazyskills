Lazyskills is a terminal UI for discovering, previewing, installing, and managing AI skills.

This documentation focuses on product behavior and feature usage.

## What to Read First

- [Overview](/docs/overview): what lazyskills is and who it is for.
- [Skills CLI Integration](/docs/skills-cli): how lazyskills delegates install/remove operations.
- [Features](/docs/features/search-discovery): the main UI capabilities.

## Quick Local Run

1. Install dependencies for this repository.
2. Run `just dev` to start the TUI.
3. Run `just docs` to preview this docs site.
