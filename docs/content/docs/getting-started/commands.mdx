---
title: Common Commands
description: Commands used by maintainers and contributors.
---

## Development

- `just dev` runs the terminal UI locally.
- `just check` compiles the Rust project without building release artifacts.

## Documentation

- `just docs` starts the docs application from `docs/`.
- `just docs-sync` regenerates the reference pages from Rust source comments.
- `just docs-sync-check` fails when generated pages are stale.
- `just docs-gh-pub` exports and publishes static docs to `gh-pages`.
