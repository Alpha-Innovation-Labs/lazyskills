---
title: Overview
description: What lazyskills is and what problem it solves.
---

Lazyskills is a terminal-first interface for working with reusable AI skills.

It helps you:

- browse skills across local and project scopes,
- inspect skill details and markdown content,
- install and remove skills safely,
- manage favorites and startup behavior.

The project is designed for users who want a fast keyboard-driven workflow instead of managing skill folders manually.
