skills-tui is a Rust terminal UI app for browsing, searching, previewing, installing, updating, and removing OpenCode skills. It renders local project/global skill trees and integrates with skills.sh APIs plus a configurable skills CLI command mode (global binary or npx). Tech stack: Rust 2021, ratatui + ratkit for TUI, crossterm input/events, reqwest blocking client, serde/serde_json, regex, anyhow.
