Primary commands:
- just dev -> run TUI locally (cargo run)
- just check -> compile check (cargo check)
- cargo check -> direct compile validation
- cargo run -> run app directly
Release/utilities:
- just pub-bump -> bump patch version in Cargo.toml and check
- just pub -> release flow (auto-commit, bump, tag, optional push, cargo publish)
Useful Darwin shell utils: git, ls, pwd, rg, awk, sed, grep, find.
