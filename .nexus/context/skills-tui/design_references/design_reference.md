# Design Reference

## Scope

This project currently focuses on terminal workflows and context artifacts. No new visual UI contract was introduced by the skills.sh discovery work.

## Design Token Inventory

- Color: inherited from terminal/theme defaults.
- Typography: terminal monospace defaults.
- Spacing: command and markdown readability conventions only.
- Radius: not applicable.
- Elevation: not applicable.
- Motion: not applicable.

## Style Contracts in Use

- Markdown structure in `.nexus/context/**` files.
- Tables for context `Next Actions` and reference summaries.

## Layout Parameters

- Single-document markdown layout.
- Flat heading hierarchy with predictable section order.

## Variants and States

- Not applicable for this documentation-only scope.
